Uma requisição é composta de 

-> URL: https://endereço.com

-> Request Method: GET | POST | PUT | DELETE | PATCH

-> Request e Hesponse Headers

-> Request Body (em certos casos)

------------------------------------
SEVIDOR PROCESSA ESSA Requisição
------------------------------------

Uma resposta do servidor é composta de

-> STATUS CODE -> significa oque aconteceu com a nossa requisição

-> Response Headers -> vai me dizer como a resposta esta configurada pro cliente saber processar ela

-> Response Body -> a resposta em si (não obrigatoria)
